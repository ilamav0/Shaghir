# 🍽️ Shaghir — Smart Restaurant & Café Reservation App

**Shaghir** is a mobile application designed to make reserving tables in restaurants and cafés easier, smarter, and more convenient for customers.

The app allows users to search and reserve places based on specific preferences such as location, parking availability, Wi-Fi, family-friendly environments, and overall atmosphere — with the ability to preview menus, photos, and customer reviews before booking.

---

## 📱 Project Overview

Finding a suitable restaurant or café can be time-consuming. **Shaghir** solves this problem by offering a smart filtering and reservation system that helps users quickly choose the best place according to their needs and preferences.

This project focuses heavily on **UX/UI principles** to ensure a smooth and intuitive user experience.

---

## ✨ Features

- 🔍 Search restaurants/cafés by:
  - Location
  - Parking availability
  - Wi-Fi availability
  - Family-friendly place
  - Environment & area

- 🪑 Table reservation system
- 📋 View menu before visiting
- 🖼️ Browse photos of the place
- ⭐ Read customer reviews
- 🎨 Clean and user-friendly interface (UX/UI focused)

---

## 🧠 UX/UI Focus

The application was designed following UX/UI best practices:

- Simple and intuitive navigation
- Clear filtering system
- Comfortable visual hierarchy
- User-centered design to reduce booking time
- Attractive and modern interface

---

## 🛠️ Technologies Used

- **Android Studio**
- **XML** for UI design
- **Java** for application logic
- UX/UI design principles

---

## 🧩 How It Works

1. User opens the app.
2. Selects preferences using filters.
3. Views matching restaurants/cafés.
4. Checks menu, photos, and reviews.
5. Reserves a table directly from the app.

---

## 🎯 Project Goal

To improve the dining experience by helping customers choose the right place faster and reserve tables effortlessly based on their personal preferences.
